Philosophy

The Evervolv project consists of people (around the world now) that want to learn how to do things related to the Android Open Source Project. It's not just a fork that has a few features added in and some themes changed, or a project that just wants to see how many devices can be ported. We've all learned something in the process of porting devices, and creating features, and continue to do so. With contributions from other forks like CyanogenMod and CodeAurora we are able to learn from them and continue even with older devices.
